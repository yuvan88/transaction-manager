# Transaction Manager (Vue.js + PHP + MySQL)

## Requirements
- XAMPP
- PHP 8+
- MySQL
- Git

## How to Run Locally

1. Clone the project
git clone <your-repo-url>

2. Move folder to:
C:/xampp/htdocs/

3. Start Apache and MySQL in XAMPP

4. Import database
- Open http://localhost/phpmyadmin
- Create database: transaction_manager
- Run the SQL to create table

5. Open in browser:
http://localhost/transaction-manager/frontend/index.html
"# transaction-manager"
